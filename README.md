# usm-vuex-demo

## Features

- TSX
- Using usm-vuex for modular design
- TypeScript
- Router
- Dynamic import with lazy loading
- Subpackage
- Monorepo
- E2E test
- Unit test
- Lint

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
